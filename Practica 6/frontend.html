Frontend

El Front end es la parte visible de una web, la que conecta con los usuarios que la visitan. Es la parte que muestra el diseño, los contenidos y permite a los visitantes navegar por las diferentes páginas mientras lo deseen12. En otras palabras, es la cara pública de un sitio web.


¿Para qué sirve el Front end?


Su función principal es permitir que los usuarios accedan a una web. Debe estar diseñado para ser fácilmente utilizable y atractivo. La experiencia de usuario depende de su usabilidad y estética. Un buen Front end puede atraer visitantes, retenerlos y generar más visibilidad para un negocio


Fundamentos del Desarrollo Front end


HTML (HyperText Markup Language): Es el lenguaje de marcado que define la estructura de una página web. Se utiliza para crear elementos como títulos, párrafos, imágenes y enlaces.


CSS (Cascading Style Sheets): Permite dar estilo a los elementos HTML. Define colores, fuentes, márgenes y otros aspectos visuales.


JavaScript: Es un lenguaje de programación que permite agregar interactividad a las páginas web. Se utiliza para crear animaciones, validar formularios y más
